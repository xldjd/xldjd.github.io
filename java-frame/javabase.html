1.java基础

1.1为什么java代码能实现一次编写、到处运行？

JVM虚拟机是跨平台的关键。程序运行前，java源码(.java)经过编译器编译为字节码文件(.class)。 在程序运行时通过jvm将字节码文件翻译成特定平台下的机器码并运行，应此，只要在不同平台安装对应的jvm，就可以运行字节码文件。

1.2一个java文件里可以有多个类吗？

一个java文件里可以有多个类，但最多只能有一个被public修饰，且这个java文件中使用public修饰的类的名称 必须和java文件名一样。

1.3对java访问权限的了解

java语言提供了三种访问修饰符，即private、protected、public，在使用这些修饰符修饰目标时， 可以产生四种访问权限、即private、default、protected、public。
♣private:该成员可以被类内部成员访问；
♣default:该成员可以被类内部成员访问,也可被同一包下其他的类访问；
♣protected:该成员可以被类内部成员访问,也可被同一包下其他的类访问,还可以被他的子类访问；
♣public:该成员可以被任意包下，任意类的成员进行访问；
在修饰类时，该类只有两种访问权限对应如下：
♣default:该类可以被同一包下其他的类访问；
♣public:该成员可以被任意包下，任意类的类进行访问；

1.4对java数据类型的了解

java数据类型包括基本数据类型和引用数据类型两大类
基本数据类型有8个，分为4个小类，分别是整数型(byte/short/int/long)、浮点型(float/double)、字符型(char)、布尔型(boolean) 其中，四个整数型中int最为常用。浮点型中double最为常用。除布尔型之外的其他7个类型，都可以看做为数字类型，它们之间 可以进行类型转换。
